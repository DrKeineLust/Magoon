### Hey, what is this, and how can I get it to work?
This is a theme created for the Chrome Extension [React Developer Tools](https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi?hl=en). You install it by doing the following:

▶ Enter the extensions settings (It's a little cogwheel)  
▶ Click the Pen Icon  
▶ Get the content of theme.json, or go to [**this link**](https://goo.gl/Gds7zy) and copy the json  
▶ Paste the content you just copied into the input bar which says "Import/export"  
▶ You can now choose the Theme in the dropdown menu
